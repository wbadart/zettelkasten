Paramorphism

A recursion scheme that tears down a recursive structure using an algebra that can view some of the original structure. From the Greek “παρά” for “beside” (as in “parallel”). Like a Catamorphism, a paramorphism can see the current result of bottom-up recursion, but can also see the “argument” or “view of the original structure” that got the computation to that point.

para :: Functor f => RAlgebra f a -> Term f -> a
para f = out >>> fmap (id &&& para f) >>> f

where

type RAlgebra f a = f (Fix f, a) -> a
newtype Fix f = In { out :: f (Fix f) }

Alternatively, the curried formulation:

type RAlgebra' f a = Fix f -> f a -> a

para' :: Functor f => RAlgebra' f a -> Fix f -> a
para' alg t = out t & fmap (para' alg) & alg t

Ed Kmett’s recursion schemes field guide characterizes paramorphism as primitive recursion. As an example, we can see how expressing factorial as a paramorphism aligns with the typical inductive definition. Given:

data NatF f = ZeroF | SuccF f deriving Functor
type Nat = Fix NatF

int :: Nat -> Int
int = cata \case
  ZeroF   -> 0
  SuccF n -> n + 1

we can see how

factorial :: Nat -> Int
factorial = para \case
  ZeroF          -> 1
  SuccF (n, fac) -> (int n + 1) * fac

lines up with

$$ \begin{split} 0! &= 1 \\ (n + 1)! &= (n + 1) * n! \end{split} $$

In the inductive case of the Haskell definition, fac is the recursive result “so far” and n is the argument that was used to compute it (i.e. factorial n == fac).

Note: the code above uses BlockArguments and LambdaCase.

In the above example, I fold inductively-defined Nats into Ints. The recursion-schemes package uses the Base type family to make this more natural.

    • A recursion scheme modeling primitive corecursion over a coinductive type; a short-circuitable unfold; an extension of Anamorphism; the categorical Dual dual to Paramorphism#. From the Greek “ἀπο” for “away from” (as in “apostasy”).

    • The composition of a Catamorphism and an Anamorphism. Hylomorphism models general recursion (in contrast to Paramorphism#, which models primitive recursion). In other words, hylomorphisms are Turing Complete.
